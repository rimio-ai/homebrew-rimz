# Homebrew tap for RimZ

Install [RimZ](https://github.com/rimio-ai/rimz) with one command:

```sh
brew install rimio-ai/rimz/rimz
```

Homebrew adds this tap automatically. Run `brew upgrade rimz` to install a newer tagged release.

The RimZ release workflow updates [`Formula/rimz.rb`](./Formula/rimz.rb) after each `v*` tag publishes its macOS archives and checksums.
